什么是算法

算法：一个计算过程，解决问题的方法

时间复杂度

用来评估算法运行效率的一个式子（单位）。
一般来说，时间复杂度高的算法比复杂度低的算法慢。
常见的时间复杂度（按效率排序）
O(1)<O(logn)<O(n)<O(nlogn)<O(n^2)<O(n^2logn)<O(n^3)

空间复杂度

用来评估算法内存占用大小的一个式子。
