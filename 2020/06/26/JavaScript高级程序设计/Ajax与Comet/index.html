第二十一章 Ajax与Comet

发表于 2020-06-26 更新于 2020-07-25 分类于 JavaScript高级程序设计

Ajax的核心是XMLHttpRequest对象（简称XHR），这是由微软首先引入的一个特性，其他浏览器提供商后来都提供了相同的实现。XHR为向服务器发送请求和解析服务器响应提供了流畅的接口，能够以异步方式从服务器取得更多信息，意味着用户单击后，可以不必刷新页面也能取得新数据。也就是说，可以使用XHR对象取得新数据，然后再通过DOM将新数据插入到页面中。

熟练使用XHR对象已经称为所有Web开发人员必须掌握的一项技能。
