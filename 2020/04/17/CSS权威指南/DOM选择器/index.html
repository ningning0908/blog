属性选择器

属性选择器是在CSS2中引入的，他们能够指定一些规则，这些规则根据元素的属性（诸如href或title）以及这些属性的值对元素进行匹配

与开始选择器相反，使用该选择器去选择以指定的值结束的属性

任意子串属性值选择器（任意选择器）

该选择器使用规则即在指定的属性字符串的内部任意位置搜索指定的子串
该选择器使用的符号是(*)
